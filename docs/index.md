# SorA

### A Homeworking Tracking App

[View log](./log)

## Info

SorA ( **S**orA: a h**o**mework T**r**acking **A**pp )
可记录历史的作业管理工具。

本 App 通过 App Inventor 2 制作。

目前共三个页面：

1. 今日：用于添加、查看、修改、标记今日作业
2. 历史：会看过去的作业
3. 更多：加载设置项和关于界面

## Pack and Export

Linux 环境，需要安装 zip 。

将本 Git `clone` 到本地后，执行：

```
$ cd library
$ zip -r SorA.aia ./*
```
之后在 App Inventor 2 中导入生成的 `SorA.aia`即可。
